# Blood-Connect
Blood Connect - A blood donation and donor management website built with HTML, CSS, and JavaScript

Project Overview

BloodConnect is a web-based blood donation management system developed using HTML, CSS, and JavaScript. It helps connect blood donors and recipients through an easy-to-use platform.

Features

- User Registration
- User Login
- Donor Registration
- Find Donor by Blood Group
- Emergency Blood Request
- Contact Form
- Local Storage Data Management

Technologies Used

- HTML
- CSS
- JavaScript
- Local Storage

Pages

- Home Page
- Login Page
- User Registration Page
- Become Donor Page
- Find Donor Page
- Emergency Request Page
- Contact Page


Objective

The objective of this project is to simplify the process of finding blood donors and provide a quick communication platform between donors and recipients.
